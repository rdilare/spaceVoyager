# spaceVoyager
Space Voyager is a game written in python.



## Play it
clone repository:
```
git clone https://github.com/rdilare/pygame-snake.git
```

```
cd pygame-snake
```

requirements(prefere using a virtual-environment):
```
pip3 install -r requirements.txt
```

run the game 
```
python3 app.py
```

- UP : w
- Down : s
- LEFT : a
- RIGHT : d
- FIRE : space
- AIM : MOUSE
