# spaceVoyager
Space Voyager is a game written in python.



## Play it
clone repository:
```
git clone https://github.com/rdilare/spaceVoyager.git
```

```
cd pygame-snake
```

requirements(prefere using a virtual-environment):
```
pip3 install -r requirements.txt
```

run the game 
```
python3 app.py
```

- w : UP
- s : Down
- a : LEFT
- d : RIGHT
- space : FIRE
- mouse : AIM
